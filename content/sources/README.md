# content/sources/

Items individuels collectés (1 fichier markdown = 1 item).

Convention de nommage : `YYYY-MM-DD-<source>-<slug>.md`

- `<source>` ∈ `github | youtube | reddit | hn | blog | manual`
- `<slug>` : slug court dérivé du titre

Frontmatter standard :

```yaml
---
source: github | youtube | reddit | hackernews | blog
url: https://...
title: ...
collected_at: 2026-05-23T07:00:00Z
tags: [coding-agent, mcp]
complexity: intermediate

---
```
